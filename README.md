## ⚙️ Setup (30 seconds)

Open **index.html** and find:

```js
const SERVER_IP = "play.yourserver.com";
const SERVER_PORT = 25565;
```

Replace with your server:

```js
const SERVER_IP = "mc.example.net";
const SERVER_PORT = 25565;
```

Done 🎉

---

## 🚀 Deploy to GitHub Pages

1. Upload `index.html` to repository
2. Open repository **Settings**
3. Go to **Pages**
4. Source → `Deploy from branch`
5. Select `main` + `/root`

Your site:

```
https://username.github.io/repository/
```

## 📡 How It Works

The page sends a request to a public Minecraft status API and updates automatically.

Browser → API → JSON → Page updates

No server required 😄
